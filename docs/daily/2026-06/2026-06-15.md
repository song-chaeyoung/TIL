# 2026-06-15 학습 일지

## 오늘 한 일

- SQL 전반 학습 (DDL·DML·TCL·DCL)
- 서브쿼리·조인·뷰·인덱스 개념 정리
- ERD 표기법 및 정규화(1NF~BCNF) 학습

## 배운 내용

### SQL 분류

- **DDL**: CREATE / ALTER / DROP / TRUNCATE — 구조 정의
- **DML**: SELECT / INSERT / UPDATE / DELETE — 데이터 조작
- **TCL**: COMMIT / ROLLBACK / SAVEPOINT — 트랜잭션 제어
- **DCL**: GRANT / REVOKE — 권한 관리

### 조인 종류

- **INNER**: 조건 만족하는 레코드만
- **LEFT OUTER**: A 전체 + 조건 만족하는 B (불일치 → NULL)
- **RIGHT OUTER**: B 전체 + 조건 만족하는 A (불일치 → NULL)
- **FULL OUTER**: A·B 전체, 불일치 측은 NULL

### 인덱스

- SELECT 성능은 향상, INSERT/UPDATE/DELETE는 오히려 저하
- 클러스터형(테이블당 1개, PK 기본 적용) vs 세컨더리(여러 개 가능)
- 데이터 충분 + 조회 빈번한 경우에만 효과적. 테이블당 3개 이하 권고

### 정규화

- **1NF**: 모든 필드가 원자값
- **2NF**: 1NF + 기본키 전체에 완전 종속 (부분 종속 제거)
- **3NF**: 2NF + 이행적 종속 제거
- **BCNF**: 3NF + 모든 결정자가 후보키
- **역정규화**: 조인 비용 줄이기 위해 의도적으로 테이블 합침
