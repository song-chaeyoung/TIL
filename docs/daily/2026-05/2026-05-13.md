# 2026-05-13 학습 일지

## 오늘 한 일

### 정처기 스터디

### COCOMO(COnstructive COst MOdel)

- 보헴 제안, LOC(원시 코드 라인 수)에 의한 비용 산정 기법
- 중소 규모 소프트웨어 프로젝트 비용 추정에 적합
- 소프트웨어 개발 유형
  - 조직형 (Organic Mode)
  - 반분리형 (Semi-Detached Mode)
  - 내장형 (Embedded Mode)

### Dos 공격

- Land Attack = 출발지 IP와 목적지 IP를 같게 조작
- Smurf = ICMP Echo 요청을 브로드캐스트 주소로 보내 피해자에게 응답 폭주
- SYN Flooding = SYN 요청을 대량으로 보내 연결 자원 고갈
- Ping of Death = 큰 패킷 또는 비정상 패킷으로 공격

### 리눅스 커널 버전 확인 명령어

- chmod = 파일 권한 변경
- ls = 파일 목록 출력
- uname = 시스템/커널 정보 확인
- rm = 파일 삭제

### E-R 다이어그램

- 개체(Entity) = 사각형
- 관계(Relationship) = 마름모
- 속성(Attribute) = 타원
- 연결 = 선

### 정규화

- 1NF = 원자값
- 2NF = 부분 함수 종속 제거
- 3NF = 이행 함수 종속 제거
- BCNF = 모든 결정자가 후보키
