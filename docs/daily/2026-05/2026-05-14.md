### CSMA/CD

- 여러 컴퓨터가 같은 통신 회선을 같이 사용할때 충돌을 처리하는 방식
- Ethernet(IEEE 802.3)이 CSMA/CD 방식의 LAN
  802.3 = Ethernet = CSMA/CD
  802.11 = Wi-Fi = CSMA/CA
  802.5 = Token Ring
  802.4 = Token Bus
  **CSMA/CD** → Collision Detection → **충돌 감지 → 유선 LAN**
  **CSMA/CA** → Collision Avoidance → **충돌 회피 → 무선 LAN**

### 카티션 프로덕트

- R × S의 차수 = R의 차수 + S의 차수
- R × S의 카디널리티 = R의 카디널리티 × S의 카디널리티

### SSH

- 다른 컴퓨터에 로그인, 원격 명령 실행, 파일 복사등을 수행할 수 있도록 다양한 기능을 지원하는 프로토콜
- 데이터 암호화와 강력한 인증 방법으로 보안성이 낮은 네트워크에서도 안전하게 통신
- key를 통한 인증방법을 사용하려면 공개키를 서버에 등록해야 한다
- 기본 네트워크 포트는 22번
