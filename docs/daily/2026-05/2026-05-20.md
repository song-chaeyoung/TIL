# 2026-05-20 학습일지

## 오늘 한 일

### 정처기 스터디

#### 정규화 설계

- 개념적 설계 : E-R 다이어그램, 개체/관계 도출
- 논리적 설계 : 릴레이션 변환, 정규화
- 물리적 설계 : 저장 구조, 접근 경로, 인덱스

#### 스레드

- 사용자 수준 스레드 = 전환 빠름, 오버헤드 적음
- 커널 수준 스레드 = 커널이 스레드별 관리 가능

#### MQTT

- IoT 환경에서 자주 쓰이는 경량 메시지 프로토콜
- TCP/IP 기반
- 발행-구독 방식

#### 블랙박스, 화이트박스 테스트

- 블랙박스 : 요구사항 명세서 기반, 명세대로 동작하는지 테스트, 동치 클래스 분해, 경계값 분석, 원인-결과 그래프, 오류 예측
- 화이트박스 : 소스코드 구조, 내부 로직 테스트

#### 보안 요소

- 기밀성 : 인가된 사람만 볼 수 있음
- 무결성 : 인가된 사람만 수정할 수 있음
- 가용성 : 필요할 때 사용할 수 있음
  그 외
  인증 : 합법적인 사용자인지를 확인하는 모든 행위, 패스워드, 인증용 카드, 지문 검사
  부인 방지 : 송/수신 증거 제공

#### MOM (Message-oriented Middleware)

- 메세지 지향 미들웨어
- 메시지 큐를 이용한 비동기 통신
- 비동기 처리, 안정적인 메시지 전달, 시스템간 느슨한 연결

## 내일 할 일
