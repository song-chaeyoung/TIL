# TIL 운영 방식

이 사이트는 OS, Network, 정보처리기사 공부 내용을 날짜별로 기록하는 학습 일지입니다.

## 기록 원칙

- 파일명은 `YYYY-MM-DD.md` 형식으로 작성합니다.
- 문서는 `docs/daily/YYYY-MM/` 아래에 둡니다.
- 각 문서에는 `title`, `description`, `tags`를 작성합니다.
- 단순 실수보다 오해한 것, 검증한 것, 다시 볼 것을 중심으로 기록합니다.

## 기본 템플릿

```md
---
title: YYYY-MM-DD 학습 일지
description: 키워드 1, 키워드 2, ...
tags: [분야1, 분야2, ...]
---
```
