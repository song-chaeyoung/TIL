# TIL

공부한 내용을 꾸준히 기록하고 정리하는 Today I Learned 저장소입니다.

## Links

- 배포 페이지: https://today-til.github.io/TIL/
- 개인 GitHub: https://github.com/song-chaeyoung
- TIL 저장소: https://github.com/TODAY-TIL/TIL

## 기록 방식

- 날짜별 학습 일지는 `docs/daily/YYYY-MM/YYYY-MM-DD.md`에 작성합니다.
- OS, Network, 정보처리기사 공부 내용을 중심으로 기록합니다.
- 단순 실수보다 오해한 것, 검증한 것, 다시 볼 것을 중심으로 남깁니다.
